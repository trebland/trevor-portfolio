# Trevor Bland — Profile

Personal profile / online resume. Built with React + Vite.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build      # outputs static site to /dist
npm run preview    # preview the production build locally
```

## Edit content

All resume content lives in [`src/data.js`](src/data.js) — name, experience,
projects, skills, and education. Edit that one file and the page updates.

## Deploy

Static site. Deployed on Vercel (build command `npm run build`, output dir `dist`).
